Lab 13: Online Class Joining Flowchart Process

Logic flow for joining a university lecture online:

  1. Start: Receive class notification.
  2. Decision: Is it class time yet? If No, wait. If Yes, click the meeting link.
  3. Process: Log in with university student portal credentials.
  4. Decision: Is the internet connection stable? If No, troubleshoot router. If Yes, enter virtual waiting room.
  5. Input/Output: Teacher admits student; video/audio streams load.
  6. End: Attend the live lecture.
